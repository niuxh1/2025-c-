Table 1. Variable Definitions
Symbol Domain Description
\(M_{cst}\) \(\mathbb{R}_{\geq 0}\) Weighted medal score for country \(c\) in sport \(s\) and year \(t\)
\(\mathcal{M}\) {Gold, Silver, Bronze} Set of medal types
\(\omega_m\) {3, 2, 1} Medal weight coefficients (Gold:3, Silver:2, Bronze:1)
\(D_{ct}\) {0, 1} Coach engagement dummy (1=active in year \(t\))
\(\mathcal{C}\) Set of countries Full sample of countries (excluding reference country)
\(\epsilon_{cst}\) \(\sim \mathcal{N}(0,\sigma^2)\) Normally distributed error term
